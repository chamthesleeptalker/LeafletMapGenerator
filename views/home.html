Make your own map using leaflet! Random project using leaflet and angular :)

Instructions:

  1. Click "This Button" to start the map magic.
  2. Pan over to your desired location while dragging the marker.
  3. Supply the needed attribute information.
  4. Click "Generate Map!" button.
  5. Copy the code.
  6. Enjoy! Haha.

Reminders:

  1. Follow the Instructions! You might catch errors that I'am too lazy to catch :P
  2. Create a div with "map" as its i.d.
  3. Add a height attribute to your "map" div. Unless, the map won't render in your page. Change the width of the div as well.
  4. CHEERS!
